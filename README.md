# ShopFlow - Inventory & Order Management System

## Live Demo

Frontend: https://shop-account-manager.netlify.app/

## Overview

ShopFlow is a full-stack business management platform designed for small and medium-sized shops to manage customers, inventory, orders, transactions, and reports efficiently.

## Features

* User Authentication
* Customer Management
* Inventory Management
* Order Tracking
* Financial Reporting
* Responsive Dashboard
* Secure REST APIs
* Pagination and Filtering

## Tech Stack

### Frontend

* React.js
* TypeScript
* Tailwind CSS

### Backend

* Node.js
* Express.js

### Database

* MongoDB
